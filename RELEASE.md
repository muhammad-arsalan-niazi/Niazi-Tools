# Niazi Tools - Version 6.5.0 Release Notes

**Niazi Tools** has been updated with two powerful new offline utilities designed to make email campaign management more flexible and precise.

## 🚀 What's New

### 1. Campaign Builder (No Timer)
We've added a stripped-down version of our popular Email Campaign Builder. This new tool allows you to effortlessly assemble campaign data (matching Emails, Subjects, and Paragraphs) without needing to configure or worry about time intervals. It's perfect for batch processing when scheduling isn't a requirement!

### 2. Campaign Builder (Custom Timer)
For those who need absolute precision over their scheduling, we've introduced the Custom Timer version. You can now upload a custom `.txt` file containing your exact time intervals (one per line). The tool enforces strict validation to ensure your custom timers perfectly match your paragraph count, preventing any data mismatches before they happen.

## 📝 Other Updates
- **Blog & SEO**: Added dedicated blog posts and automatically generated FAQs for both new tools to improve discoverability.
- **Sitemap Optimization**: The new tools are dynamically integrated into the application's sitemap.
- **Documentation**: Updated the README to reflect the expanded toolkit (now boasting 20 powerful utilities!).

All tools remain 100% free, private, and process data entirely within your browser!
